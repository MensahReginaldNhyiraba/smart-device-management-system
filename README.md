## Project Description

This project is a Smart Device Management System developed using Python. It demonstrates the use of Object-Oriented Programming concepts including classes, objects, inheritance, encapsulation, constructors, getters and setters, and the use of `super()`.

## Features

- Display device information
- Turn devices ON and OFF
- Read temperature from a sensor
- Increase and decrease light brightness
- Start and stop camera recording
- Menu-driven interface

## Programming Concepts Used

- Classes and Objects
- Inheritance
- Encapsulation
- Constructors
- Getters and Setters
- Method Overriding
- `super()`

## How to Run

1. Open the project in PyCharm.
2. Open `main.py`.
3. Click **Run** or press **Shift + F10**.
4. Use the menu displayed in the console.
